# Sample Certificate Template Instructions

## How to Create Certificate Templates

### Option 1: Using Canva (Recommended)

1. Go to https://www.canva.com
2. Search for "certificate"
3. Customize with your:
   - Organization name
   - Colors and branding
   - Text fields
4. Export as PDF (1200x850px)
5. Upload via admin dashboard

### Option 2: PowerPoint/Google Slides

1. Create 1200x850px presentation
2. Design certificate layout
3. Leave spaces for:
   - {{STUDENT_NAME}}
   - {{COURSE_NAME}}
   - {{DATE}}
   - {{SIGNATURE}}
4. Export as PDF
5. Upload via admin dashboard

### Option 3: Figma

1. Create 1200x850px artboard
2. Design with proper typography
3. Export as PDF
4. Upload via admin dashboard

## Best Practices

- Use professional fonts
- Include your logo/branding
- Leave space for student details
- Use high quality (300 DPI)
- Test template with sample student
- Keep file size under 5MB

## Fields Automatically Added

- Certificate ID
- Student Name
- Course Name
- Issue Date
- Completion Date
- QR Code (optional)
- Verification Link

## Sample Template Specs

- Dimensions: 1200x850px
- Format: PDF or PNG
- DPI: 300 (for printing)
- Max Size: 5MB
- Colors: RGB or CMYK
